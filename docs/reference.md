# Markdown Reference

Simplified version of basic syntax. For more information, visit [Markdown Guide](https://www.markdownguide.org)

---

## Headings

```
# H1
## H2
### H3
...
###### H6
```

## Text styles

`**bold**`

`*italic*`

## Lists

Ordered list:

```
1. First item
2. Second item
3. Third item
```

Unordered list:

```
* List item
* List item
* List item
```

## Links & Images

`![Image Alt Text](image.jpg)`

`[Link Text](link.com)`

## Tables

```
First Header  | Second Header
------------- | -------------
Content Cell  | Content Cell
Content Cell  | Content Cell
```

## Code & Blockquotes

```
`code`
```

`> blockquote`
